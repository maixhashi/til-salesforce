## 概要

Salesforce Platform Developer I（PD1）は、Lightning Platformでのカスタムアプリケーション開発スキルを証明する認定資格である。

宣言的機能とプログラミング的機能（Apex、Visualforce）の両方を理解し、適切な方法を選択してビジネス要件を実装できる能力が求められる。

## 試験の出題範囲と配点

試験は4つの主要分野から構成される。

```mermaid
flowchart TB
    subgraph exam["Platform Developer I 試験構成"]
        direction TB
        A["Developer Fundamentals<br/>開発者の基礎<br/>27%"]
        B["Process Automation and Logic<br/>プロセス自動化とロジック<br/>28%"]
        C["User Interface<br/>ユーザーインターフェース<br/>25%"]
        D["Testing, Debugging, and Deployment<br/>テスト・デバッグ・デプロイ<br/>20%"]
    end

    A --> total["合計: 100%"]
    B --> total
    C --> total
    D --> total
```

| 分野 | 配点 | 主な内容 |
|------|------|----------|
| Developer Fundamentals | 27% | データモデリング、SOQLクエリ、DML操作 |
| Process Automation and Logic | 28% | Apexトリガー、フロー、プロセスビルダー |
| User Interface | 25% | Visualforce、Lightning Web Components |
| Testing, Debugging, and Deployment | 20% | Apexテスト、デバッグ、変更セット |

## 試験の詳細

| 項目 | 内容 |
|------|------|
| 問題数 | 60問（+5問のスコア対象外問題） |
| 合格ライン | 68% |
| 問題形式 | 多肢選択式・複数選択式 |
| 試験時間 | 105分 |
| 受験料 | $200 USD（再受験: $100 USD） |
| 前提資格 | なし（ただし実務経験推奨） |

## 必要なスキル

```mermaid
flowchart LR
    subgraph declarative["宣言的開発"]
        D1["フロー"]
        D2["プロセスビルダー"]
        D3["数式項目"]
        D4["入力規則"]
    end

    subgraph programmatic["プログラム的開発"]
        P1["Apex"]
        P2["SOQL/SOSL"]
        P3["Visualforce"]
        P4["LWC"]
    end

    declarative --> choice{"適切な手法の<br/>選択"}
    programmatic --> choice
    choice --> solution["ビジネス要件の<br/>実装"]
```

主要なスキル要件:

- **Apex**: クラス、トリガー、バッチ処理、スケジュール実行
- **SOQL/SOSL**: データクエリの最適化、ガバナ制限の理解
- **Visualforce**: ページ作成、コントローラー拡張
- **Lightning Web Components**: 基本的なコンポーネント開発
- **データモデリング**: オブジェクト関係、マスター詳細/参照関係

## 維持要件

認定資格を維持するには、年に3回のSalesforceリリース（Spring、Summer、Winter）に合わせてメンテナンスモジュールを完了する必要がある。

- メンテナンスは無料
- Trailhead上で完了
- 期限内に完了しないと資格が失効
