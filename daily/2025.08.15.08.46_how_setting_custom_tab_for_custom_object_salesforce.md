## カスタムタブとは

カスタムタブは、カスタムオブジェクトにアクセスするためのナビゲーション要素です。Lightning Experienceやクラシックビューの両方で、ユーザーがオブジェクトのデータにアクセスできるようになります。

```mermaid
flowchart LR
    A[カスタムオブジェクト] -->|タブを作成| B[カスタムタブ]
    B --> C[Lightning アプリケーション]
    B --> D[クラシック アプリケーション]
    C --> E[ナビゲーションバーに表示]
    D --> F[タブバーに表示]
    
    style A fill:#e1f5ff
    style B fill:#ffe1e1
    style C fill:#e1ffe1
    style D fill:#e1ffe1
```

## カスタムタブ作成の前提条件

### 必要な権限
- 「アプリケーションのカスタマイズ」権限
- カスタムオブジェクトが既に作成されていること

### 確認事項
1. カスタムオブジェクトのAPI名を確認
2. 使用するユーザープロファイルを把握
3. 配置するアプリケーションを決定

## カスタムタブの作成手順

```mermaid
flowchart TD
    A[設定] --> B[ユーザインターフェース]
    B --> C[タブ]
    C --> D[カスタムオブジェクトタブ]
    D --> E[新規作成]
    E --> F[オブジェクトを選択]
    F --> G[タブスタイルを選択]
    G --> H[タブの表示設定]
    H --> I[アプリケーションへの追加]
    I --> J[保存]
    
    style A fill:#e1f5ff
    style E fill:#ffe1e1
    style J fill:#e1ffe1
```

### 詳細手順

#### 1. 設定画面へのアクセス
1. 設定（歯車アイコン）をクリック
2. 「設定」を選択

#### 2. タブメニューへの移動
1. クイック検索で「タブ」を検索
2. 「タブ」を選択

#### 3. カスタムオブジェクトタブの新規作成
1. 「カスタムオブジェクトタブ」セクションで「新規」をクリック
2. オブジェクトドロップダウンから対象のカスタムオブジェクトを選択

## タブスタイルの設定

### タブスタイルの要素

```mermaid
flowchart TD
    A[タブスタイル] --> B[色の選択]
    A --> C[アイコンの選択]
    B --> D[16色から選択可能]
    C --> E[200以上のアイコン]
    
    subgraph "プレビュー"
        F[選択した色とアイコンの組み合わせ]
    end
    
    D --> F
    E --> F
    
    style A fill:#ffe1e1
    style F fill:#e1ffe1
```

### 設定項目
1. **タブスタイル**: 色とアイコンの組み合わせを選択
2. **スプラッシュページのカスタムリンク**: オプション（初回アクセス時の説明ページ）
3. **説明**: タブの説明（任意）

## プロファイルへの表示設定

### タブの表示設定オプション

```mermaid
flowchart LR
    A[タブの表示設定] --> B[デフォルトで表示]
    A --> C[デフォルトで非表示]
    A --> D[タブを非表示]
    
    B --> E[すべてのユーザーに表示]
    C --> F[「すべてのタブ」から追加可能]
    D --> G[ユーザーには見えない]
    
    style A fill:#ffe1e1
    style B fill:#e1ffe1
    style C fill:#fff5e1
    style D fill:#e1e1ff
```

### プロファイル別設定
1. 各プロファイルに対して以下を選択：
   - **デフォルトで表示**: タブが常に表示される
   - **デフォルトで非表示**: ユーザーが手動で追加可能
   - **タブを非表示**: タブへのアクセス不可

## アプリケーションへの追加

### アプリケーション選択画面

```mermaid
flowchart TD
    A[アプリケーションへの追加] --> B{アプリケーション選択}
    B --> C[営業]
    B --> D[サービス]
    B --> E[マーケティング]
    B --> F[カスタムアプリ]
    
    C --> G[チェックボックスで<br/>含めるアプリを選択]
    D --> G
    E --> G
    F --> G
    
    style A fill:#ffe1e1
    style B fill:#fff5e1
    style G fill:#e1ffe1
```

### 追加手順
1. 表示させたいアプリケーションをチェック
2. 「保存」をクリック

## トラブルシューティング

### よくある問題と解決方法

#### タブが表示されない場合

```mermaid
flowchart TD
    A[タブが表示されない] --> B{原因の確認}
    B --> C[プロファイル設定]
    B --> D[アプリケーション設定]
    B --> E[権限設定]
    
    C --> F[プロファイルでタブを<br/>「デフォルトで表示」に変更]
    D --> G[アプリケーションビルダーで<br/>タブを追加]
    E --> H[オブジェクトの参照権限を確認]
    
    style A fill:#ffe1e1
    style B fill:#fff5e1
    style F fill:#e1ffe1
    style G fill:#e1ffe1
    style H fill:#e1ffe1
```

### チェックリスト
1. ✓ カスタムオブジェクトに対する参照権限があるか
2. ✓ プロファイルでタブが表示設定になっているか
3. ✓ 使用しているアプリケーションにタブが含まれているか
4. ✓ Lightning Experienceの場合、アプリケーションビルダーで設定されているか
